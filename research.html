Research Overview

My research focuses on developing ground-based optical instruments and their data reduction pipelines, improving their automation and ease-of-use. I am interested dveloping new skills and learning to apply machine learning techniques to optimise instrument performance.

Research Areas

Machine Learning Applications

Developing reinforcement learning models to optimise the overhead time between observing fields when arranging optical fibres.

Dataflow develop

Integrating modern spectroscopic data reduction packages to produce data reduction pipelines that are seamless for the end user.

Image processing

Processing high-quality image data from JWST to search for extreme objects such as black holes

Observing support

Supporting Graduate stduents and Postdocs to prepare for their observing runs using the Magellan telescope. This includes data reduction support with custom pipelines.

The LLAMAS Data Reduction Pipeline

I am developing a data reduction pipeline for the LLAMAS IFU instrument using PypeIt, an open-source spectroscopic data reduction package. The pipeline automates calibration, spectral extraction, and cube reconstruction to produce science-ready IFU data cubes. While the code is available on GitHub, it is still actively being developed and refined. The methodology includes bias and flat-field corrections, wavelength calibration, optimal extraction of spectra, and flux calibration, ensuring accurate and reliable data products for scientific analysis.

Expected Outcomes

The expected outcome of the LLAMAS data reduction pipeline is a fully automated and efficient workflow that produces science-ready IFU data cubes. These data cubes will be carefully calibrated, with bias and flat-field corrections applied, precise wavelength and flux calibration, and optimally extracted spectra.

The GitHub repository for the LLAMAS data reduction pipeline is still in active development and subject to frequent updates as new features and improvements are implemented.

Current Research Project 2

This project focuses on the development of machine learning algorithms for predictive analysis in my field of study. We are working on creating models that can accurately predict outcomes based on complex input data.

Key aspects of this research include:

  • Development of predictive models using various machine learning techniques
  • Feature selection and engineering for improved model performance
  • Addressing challenges of model interpretability and explainability
  • Application of the models to real-world datasets

Methods

We employ a range of machine learning techniques, including ensemble methods, neural networks, and probabilistic models. Our approach emphasizes not only prediction accuracy but also model interpretability, which is crucial for scientific applications.

Expected Outcomes

This research will result in improved predictive models for scientific applications, contributing to both methodological advances in machine learning and practical tools for scientific discovery. The insights gained from this work will inform future research directions in predictive modeling.
